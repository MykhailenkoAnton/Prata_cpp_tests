Що таке шаблонний клас і шаблонна функція?
/
В первую очередь это основа обобщенного програмирования
Шаблонный класс = класс, который придназначен для кодирования обобщенных алгоритмов, без приязки
к определенным параметрам
Шаблонная функция = функция, поведения которой подходит для обьектов разных типов
------------------------------------------------------------------------------------------------------------
Що таке конструктори? Які типи знаєте? С++
/
Конструктор - уникальный метод в языке с++, который не имеет типа возврата и ися конструктора == имени класса
и который автоматически вызывается при создании обьектах
конструкторы могут быть вызваны явно и неявно
если вы не определили конструктор по умолчанию для класса, компилятор это сделает за вас
иначе если вы определили конструктор, то будет вызывается этот(эти) констукторы
констуктор который не имеет параметров = конструктор по умолчанию
есть конструктор определенный пользователем, со списком параметром или без него
констукторы бывают
- по умолчанию (вызываться могут явно или неявно)
- с параметрам
конструктор копирования (аргументом обьязательно const & Constructor a)
конструктор переноса (аргументом обьязательно Constructor&& a)
------------------------------------------------------------------------------------------------------------
Чи може конструктор бути шаблонною функцією?
/
Конструктор по-умолчанию не может быть шаблонным

------------------------------------------------------------------------------------------------------------
Чи може віртуальна функція бути шаблонною?
/
В С++ шаблон функции не является функцией, поэтому шаблон не может быть виртуальным.
------------------------------------------------------------------------------------------------------------
Що таке інстанціація шаблону?
/
Процесс создания конкретного класса из шаблона путем подстановки аргументов называется
инстанцированием шаблона. В результате инстанцирования появляется специализация шаблона, 
являющаяся определённой функцией в сегменте кода или определённым пользовательским типом.
------------------------------------------------------------------------------------------------------------
Що таке спеціалізація шаблону? Часткова спеціалізація шаблону?
/
Частичная специализация шаблона — механизм языка программирования C++, 
предназначенный для специализации обобщённых шаблонных 
классов под конкретные задачи или под конкретное подмножество своих параметризованных типов данных
------------------------------------------------------------------------------------------------------------
Розкажіть про імплементацію шаблоны класів у срр-файлі?
/
Самый простой вариант помещение реализации шаблонов в однов файле с определением
/
Еще один альтернативный вариант — использовать подход трех файлов:
Определение шаблона класса хранится в заголовочном файле.
Определения методов шаблона класса хранятся в отдельном файле .cpp.
Затем добавляем третий файл, который содержит все необходимые нам экземпляры шаблона класса.
